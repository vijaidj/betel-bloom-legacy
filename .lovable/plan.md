

# Sri Sholavandan Heritage Vetrilai — Premium Website

## Overview
A single-page premium branding website for a GI-certified betel leaf business, focused on B2B, wedding bulk orders, and retail enquiries. Elegant, minimal design with deep green + gold + cream color palette.

---

## Design System
- **Primary**: Deep betel leaf green (#0F3D2E)
- **Accent**: Gold (#C8A951)
- **Background**: Cream (#F6F3E8)
- **Headings**: Premium serif font (Playfair Display)
- **Body**: Clean sans-serif (Inter)
- **Subtle betel leaf pattern** as background texture using CSS/SVG

---

## Page Sections (Single Page with Smooth Scroll)

### 1. Navigation Bar
- Brand name/logo with gold accent
- Nav links: Home, About, Products, Wedding Bulk, Contact
- WhatsApp CTA button in nav
- Mobile hamburger menu

### 2. Hero Section
- Full-width with subtle betel leaf background texture
- **Title**: "GI Certified Sholavandan Betel Leaf"
- **Subtitle**: "Premium Bulk Supply for Weddings & Puja"
- Two buttons: **Enquire Now** (scrolls to contact) + **WhatsApp Order** (opens WhatsApp)

### 3. Highlights Section
- 4 elegant icon cards in a row:
  - 🌿 Farmer Direct
  - 🏅 GI Focused
  - 📦 Bulk Orders
  - 🌱 Fresh Harvest
- Minimal icons with gold accents

### 4. About Section
- Short paragraph about being native farmers from Sholavandan
- Traditional cultivation heritage
- Premium quality commitment
- Optional decorative betel leaf illustration

### 5. Products Section
- Two product cards with elegant styling:
  - **Fresh Betel Leaves** — description + "Enquire" CTA
  - **Betel Leaf Powder (Herbal Tea)** — description + "Enquire" CTA

### 6. Wedding Bulk Section
- **Title**: "Bulk Betel Leaves for Weddings & Ceremonies"
- Short description about bulk supply capabilities
- **Request Quote** button (scrolls to contact form)
- Elegant gold border/accent styling

### 7. Contact / Enquiry Section
- Clean form with fields: Name, Phone, City, Quantity, Purpose (dropdown), Message
- Submit button (initially opens WhatsApp with pre-filled message; future-ready for API)
- Input validation using Zod

### 8. Footer
- Contact details (phone, email)
- WhatsApp link
- Location: Sholavandan, Tamil Nadu
- Simple copyright line

### 9. Floating WhatsApp Button
- Fixed bottom-right WhatsApp icon button
- Opens WhatsApp with pre-filled greeting message

---

## Technical Details
- Fully mobile responsive
- Smooth scroll navigation
- Google Analytics placeholder (gtag script ready)
- SEO meta tags and Open Graph tags
- No e-commerce or cart functionality
- Form data sent via WhatsApp link (no backend needed for MVP)
